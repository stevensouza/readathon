Verification Boxes - Consistent Styling

All verification boxes with unified font sizes and styling

Consistent Font Size System
Goal: Create visual consistency across all verification boxes using a standardized typography system. All boxes use the same font sizes for comparable elements (main values, labels, subtitles, secondary values).

Typography Style Guide

  • Main Value: 2rem (white, 900 weight) - The primary number/metric
  • Title/Label: 1rem (85% white, 700 weight, uppercase) - Describes the metric
  • Subtitle: 0.9rem (75% white, 600 weight) - Additional context
  • Secondary Value: 1.5rem (white, 900 weight) - Related metrics
  • Secondary Label: 0.8rem (70% white, 700 weight, uppercase) - Secondary metric labels
  • Timestamp: 0.7rem (65% white, italic) - Metadata

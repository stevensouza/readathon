Enhanced Participation Metrics - Layout Options

Compare three different approaches to displaying enhanced participation data

Enhancement Context

New Metrics to Display:

  • Participated All Days: Count of readers who logged minutes every single day
  • Met Goal All Days: Count of readers who met their daily goal every single day
  • Current Participation Rate: Percentage of readers who logged any minutes

Challenge: How to best integrate these new metrics with the existing participation box without cluttering the UI?

Layout A: Expanded Blue Box
Approach: Keep all participation metrics together in one expanded verification box. Main participation rate stays prominent, with new metrics as secondary values below.
Best for: Users who want all participation data centralized in one location.

Advantages:
  • All participation data in one place - easy to find
  • Maintains existing verification box style
  • Clear hierarchy: overall rate → detailed breakdowns
  • Minimal changes to existing layout
Considerations:
  • Box becomes taller and more information-dense
  • May feel cramped with too many numbers
  • Secondary metrics may compete for attention
Layout B: Side-by-Side Boxes
Approach: Split participation metrics into three separate verification boxes. Each metric gets equal visual weight and prominence.
Best for: Users who want to emphasize all three participation metrics equally.

Layout C: Stat Cards Row
Approach: Keep the original participation verification box unchanged. Add a new row of lightweight stat cards below to show the enhanced metrics.
Best for: Users who want to maintain the current layout while adding supplementary details.

Recommendation Considerations

Choose Layout A if:

  • You want minimal changes to existing design
  • Screen space is at a premium
  • Users expect all participation data together

Choose Layout B if:

  • You want to emphasize all three participation metrics equally
  • You have horizontal screen space available
  • Visual separation is more important than compactness

Choose Layout C if:

  • You want to preserve the current participation box exactly as-is
  • You plan to add more supplementary metrics in the future
  • You like the idea of a visual hierarchy (main boxes → detail cards)
  • You want flexibility to show/hide enhanced metrics easily
